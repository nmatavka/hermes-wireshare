WireShare for Android

WireShare Android is a Team Hermes mobile shell based on the FrostWire Android codebase, reworked under the WireShare package and application identity.

This first Android milestone keeps the proven torrent UI and jlibtorrent runtime in place while the Gnutella backend is reintroduced behind protocol-neutral Android adapters.

Attribution

The Android UI, torrent runtime, Apollo player integration, and common search/transfer components were derived from the GPL-licensed FrostWire Android project. Original author and contributor notices are retained in the copied source files.

All BitTorrent networking in this Android shell is performed through frostwire-jlibtorrent, the Java wrapper around libtorrent-rasterbar.

Apollo player components remain attributed to their original upstream authors.
